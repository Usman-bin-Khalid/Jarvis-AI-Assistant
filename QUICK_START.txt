TROUBLESHOOTING:

❓ Page won't load?
   → Wait 10 seconds
   → Check if "pnpm dev" is running
   → Refresh page (Ctrl+R or Cmd+R)

❓ Microphone not working?
   → Allow microphone permission when asked
   → Check browser microphone settings
   → Try different browser

❓ AI not responding?
   → Check GOOGLE_GENERATIVE_AI_API_KEY in .env.local
   → Restart server: pnpm dev

❓ No voice output?
   → Check ELEVENLABS_API_KEY in .env.local
   → Check volume is not muted
   → Check speakers work

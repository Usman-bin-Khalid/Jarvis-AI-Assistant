   http://localhost:3000/auth/sign-up

   - Choose a password (6+ characters)

   - Ready to use! Click microphone to talk

   http://localhost:3000/auth/login

   - Click "Login"
   - Redirected to JARVIS

🆘 TROUBLESHOOTING:
================================================================================

Problem: Email not arriving
Solution: 
  - Check Spam/Promotions folders
  - Wait 2-3 minutes
  - Try different Gmail account
  - See AUTH_FIX_COMPLETE.md for detailed steps

Problem: "Unauthorized" error on confirmation link
Solution:
  - This is FIXED now - should not happen
  - If it does, link may have expired (24h limit)
  - Create new account to get new link

Problem: Page slow after clicking email link
Solution:
  - Normal if Supabase server busy
  - Just wait 5-10 seconds
  - Page will load and redirect

Problem: Can't login after confirming email
Solution:
  - Use exact same email
  - Use exact same password
  - Clear browser cookies: Ctrl+Shift+Delete
  - Try different browser

Problem: "Email not confirmed" error
Solution:
  - This is correct - you haven't clicked confirmation link
  - Go to Gmail, click the Supabase email link
  - After clicking, try login again
